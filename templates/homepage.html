What is Aquaquidado?

The project is a control system that sustain oxygen, salinity, and temperature level that minimize the mortality rate of the post larvae vannamei shrimp. The electricity source of the control system are solar panel, and a plug-in for versatility. The project will be monitored by hardware and application. The monitoring system can display the oxygen, salinity, and temperature level of the control system.
